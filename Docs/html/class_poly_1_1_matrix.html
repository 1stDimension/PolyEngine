Performs decomposition of the matrix to separate translation, rotation and scale.

This method will assert if the decomposed matrix contains non-linear factors (like projection or skew). In such case, use the other overload.

Parameters
translationMutable reference to a vector that will be set to the decomposed transaltion value.
rotationMutable reference to a quaternion that will be set to the decomposed rotation value.
scaleMutable reference to a vector that will be set to the decomposed scale value.
Returns
True if decomposition was sucesfull, false otherwise.
