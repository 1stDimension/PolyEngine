Detailed Description

template<eAllocationStrategy TYPE = eAllocationStrategy::DEFAULT>
class Poly::BaseObjectLiteralType< TYPE >

This class is variation of BaseObject that does not introduce virtual destructor.

It should be used only for constexpr classes and other PODs that do not require virtual destructors.

BaseObjectLiteralType can be templated to allow different allocation strategies.
