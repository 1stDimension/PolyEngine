Detailed Description

template<eAllocationStrategy TYPE = eAllocationStrategy::DEFAULT>
class Poly::BaseObject< TYPE >

This class is base for almost every object in the engine. It ensures that all memory management is done via custom allocators.

For safety reasons BaseObject introduces virtual destructor, to ensure that no memory leaks are ever introduced. If this is unaceptable, please refere to BaseObjectLiteralType.

BaseObject can be templated to allow different allocation strategies.
