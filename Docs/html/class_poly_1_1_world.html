◆ IterateComponents()

template<typename PrimaryComponent , typename... SecondaryComponents>
IteratorProxy<PrimaryComponent, SecondaryComponents...> Poly::World::IterateComponents ( )
inline

Allows iteration over multiple component types. Iterator dereferences to a tuple of component pointers.

To get the component out of the tuple use std::get() e.g.

std::get{YourComponentType*}(components)

If you have a C++17-compliant compiler, you can use structured bindings e.g.

for(auto [a, b] : world->IterateComponents{ComponentA, ComponentB}())
Parameters
PrimaryComponentAt least one component type must be specified
SecondaryComponentsAdditional component types (warning: returned pointers might be null!)
Returns
A proxy object that can be used in a range-for loop.
