Detailed Description

template<typename T>
class Poly::Queue< T >

Queue is a vector based container thet allocates its memory in one, continous block and provides API that is very fast when instering/popping from back/front.

This should be the goto container for any queueing purposes, like FIFO or LIFO. It does not provide API for insertion in any position.
